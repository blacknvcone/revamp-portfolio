# Dani Prasetya – Personal Portfolio

Welcome to my personal portfolio website! This project showcases my professional experience, skills, and selected projects as a Software Engineer specializing in Golang and JavaScript.

## 🚀 Project Overview
This is a modern, responsive portfolio built with [Next.js](https://nextjs.org/), [React](https://react.dev/), and [Material UI](https://mui.com/). It highlights my background, work experience, education, certifications, and projects in a clean, interactive format.

## ✨ Features
- **Hero Section**: Brief introduction, CV download, and contact button.
- **About Page**: Detailed work experience, skills, education, and certifications.
- **Projects Page**: Showcases selected projects with descriptions and links.
- **Profile Sidebar**: Personal info, navigation, and theme toggle (dark/light mode).
- **Responsive Design**: Optimized for both desktop and mobile devices.
- **Sticky Table of Contents**: Easy navigation on the About page.

## 🛠️ Tech Stack
- **Framework**: Next.js 15+, React 19
- **UI**: Material UI, Tailwind CSS, Emotion
- **State Management**: React Context
- **Deployment**: Vercel (recommended)

## 🧑‍💻 Getting Started
Clone the repo and install dependencies:

```bash
git clone https://github.com/yourusername/revamp-portfolio.git
cd revamp-portfolio
npm install # or yarn or pnpm
```

Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

## 🚢 Deployment
